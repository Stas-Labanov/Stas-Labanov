## Who I am

I create production-level full-stack applications that are useful for humanity using the modern stack. 
If it works fast and feels right, then I probably created it

## Projects

- **[Smart warehouse](https://github.com/Stas-Labanov/prosto-market)** - A virtual warehouse for your business to manage all product categories and quickly calculate revenue and expenses, **with AI** (TypeScript + Next.js + TailwindCSS)
- **[LanguageTest](https://github.com/Stas-Labanov/language-test-fullstack)** - An application for determining the language level, including grammar, listening, and translation tests (React + TypeScript + Nest.js)
- **[AITeacher](https://github.com/Stas-Labanov/telegram-mini-app_ai-teacher)** - AI teacher for learning English with integration into the Telegram messenger  **[backend part](https://github.com/Stas-Labanov/telegram-mini-app_ai-teacher-backend)** (React + TypeScript + Telegram Mini App + TailwindCSS)
- **[AdminPanel](https://github.com/Stas-Labanov/admin-panel_for_ai-teacher)** - Admin panel for AITeacher and LanguageTest, for adding new tests, words, and audio, etc. (TypeScript + React + Nest.js). 

## Stack

- **Testing:** Jest, React Testing Library
- **Tools:** Git, GitHub, GitLab, Docker, Postman
